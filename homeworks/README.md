## Домашнее задание 1. Git, Github, Github Actions. 

* Работа с Git: создание коммитов, ветвление, разрешение конфликтов, использование тегов;
* Использование GitHub: создание репозитория, настройка приватности, добавление коллабораторов, работа через веб-интерфейс;
* Основы CI/CD: написание `workflow` на GitHub Actions, автоматическая проверка работоспособности кода при push;
* Защита веток: настройка правил слияния с обязательным прохождением тестов.

## Домашнее задание 2. scraping, crawling, API, telegram bots API. 

* Парсинг статических и динамических веб-страниц (`requests`, `BeautifulSoup`, `scrapy`, `selenium`);
* Работа с REST и GraphQL API (ENCODE, Ensembl), сопоставление биологических идентификаторов;
* Настройка GitHub Actions с использованием секретов и отправка уведомлений через Telegram Bot API;
* Разработка REST API на `FastAPI` с валидацией (`pydantic`), документацией и логикой предсказания.

## Домашнее задание 3. Computer networks, protocols, server-client architecture.

* Анализ маршрутизации (`traceroute`);
* Реализация клиент-серверного протокола поверх UDP (с синхронным и асинхронным серверами через `sockets` и `asyncio/aiohttp`);
* Работа с DNS через `dig` (`A`, `CNAME`, `NS`, рекурсивное разрешение);
* Работа с ARP-кэшем и определение производителя по MAC-адресу;
* Шифрование и проверка подлинности с `OpenSSL` (AES + RSA-подпись).

## Домашнее задание 4. Контейнеризация с Docker, Apptainer.

* Работа с docker-контейнерами: создание, мониторинг, логирование;
* Использование cron-планировщика задач;
* Настройка сети и коммуникации контейнеров по API;
* Практика деплоя сервиса и обращения к нему.

## Домашнее задание 5. Пайплайны обработки данных с Airflow.

* Разработка одноразового и периодического запуска пайплайна;
* Работа с разными типами операторов в `airflow`;
* Подвязка сторонних сервисов к пайплайну, обращение с БД;
* Базовое представление результатов в `streamlit`.
